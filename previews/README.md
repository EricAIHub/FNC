## 🌟 为什么推荐将本项目下载到本地使用 Obsidian？

在使用 Obsidian 打开本仓库（Vault）后，您将获得以下三大核心能力：

### 1. 完美的 LaTeX 数学公式渲染
不同于 GitHub 网页端偶尔出现的公式乱码，Obsidian 配合 MathJax 插件可以完美呈现所有复杂的矩阵分解与误差推导公式，为您提供纸质书般的视觉质量。

### 2. 动态知识图谱 (Graph View)
利用 Obsidian 的双向链接（Backlinks），您可以清晰地看到：
- **条件数 (Conditioning)** 是如何影响 **ResNet** 稳定性的。
- **浮点数误差** 是如何通过 **迭代算法** 累积的。
- 所有的知识点不再是孤岛，而是一张逻辑严密的网。

### 3. 代码与笔记的一体化
您可以直接在笔记侧边开启终端运行 Python 脚本，实时验证“灾难性相消”或“梯度爆炸”的数值实验，真正做到“所学即所得”。

2. 解压后，打开 **Obsidian** 软件。
3. 选择 **Open folder as vault**，定位到解压后的文件夹。
